# Automation Module

> This module is part of a DevOps-focused Linux automation toolkit designed for real-world operational use cases.

## Overview
This module contains scripts to automate routine system administration tasks such as backups.

---

## Script: auto_backup.sh

### Description
Creates a compressed backup of a given directory and stores it in a specified location with a timestamp.

---

### Usage
./auto_backup.sh <source_directory> <backup_directory>

---

### Example
./auto_backup.sh /etc /home/cisco/backups

### Output
- Creates a .tar.gz file
- Adds timestamp to backup file
- Shows success or failure message

### DevOps Use Case
- Automated server backups
- Cron-based scheduled backups
- Disaster recovery setup
